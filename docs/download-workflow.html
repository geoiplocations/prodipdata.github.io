How monthly release files, passwords, and large ALL packages are published.

Monthly release model

GeoIP Locations publishes monthly releases identified by YYYY-MM. Each release includes package indexes for CSV, Parquet, and MMDB.

Package passwords

Each package index shows the access password associated with that package and format. CSV, Parquet, and MMDB can have different passwords.

Country packages

Country packages are published in the repository release folder using the pattern Export/release_YYYY-MM/FORMAT/iso=XX.zip.

ALL packages

The ALL scope files can be larger than standard repository limits. For that reason, the ALL packages are published as GitHub Release assets while the country packages remain available through the release folder.
